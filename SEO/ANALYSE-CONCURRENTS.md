# ANALYSE CONCURRENTS — Garden Expresse

Sources utilisées : accès direct aux sites concurrents quand disponible + SERP/stratégie existante dans le dossier. Google a bloqué l'accès automatisé direct aux SERP par anti-bot ; je n'ai donc pas inventé de positions Google exactes. Les concurrents ci-dessous sont ceux identifiés comme présents/pertinents sur les requêtes transport Maroc-Europe, frigorifique, groupage, transitaire/freight forwarding.

## Observations techniques extraites

### transporteur-maroc.com

- Statut : 200.
- CMS : WordPress détecté.
- Title : `Accueil - Transporteur International Maroc`.
- H1 : `Transport International Routier Maroc`.
- JSON-LD : présent.
- Forces : correspondance exacte avec mots-clés génériques.
- Faiblesses : contenu et UI peu différenciants ; opportunité Garden Expresse : produire pages plus propres, plus rapides, avec meilleur maillage et conversion.

### transport-france-maroc.com

- Statut : 200.
- CMS : WordPress détecté.
- Title : `Tarifs Transport France Maroc / Tél :+212****2300`.
- H1 : non détecté dans l'échantillon HTML.
- JSON-LD : présent.
- Forces : domaine exact sur corridor France-Maroc.
- Faiblesses : SEO on-page perfectible, crédibilité marque faible. Garden Expresse peut dépasser en qualité de contenu, structure, preuves et UX mobile.

### sobetrac.com

- Statut : 200.
- Title : `SOBETRAC - Transport Routier International Maroc Europe | Tanger`.
- Meta description claire avec fret, traction, frigorifique, devis.
- H1 détecté en doublon : `SOBETRAC — Transport Routier International Maroc Europe`.
- JSON-LD : 3 blocs.
- Forces : positionnement Tanger/Maroc-Europe crédible.
- Faiblesses : site court, très peu de liens détectés. Opportunité : Garden Expresse peut bâtir une architecture plus profonde.

### attransnorthafrica.fr

- Statut : 200 via `www`.
- HTML échantillon très pauvre : title, description, H1 non détectés.
- Forces : niche frigorifique North Africa.
- Faiblesses : si le rendu dépend du JS, Google peut moins bien interpréter. Opportunité : pages HTML statiques riches sur frigorifique Maroc-Europe.

### grlogistique.com

- Statut : 200.
- CMS : WordPress détecté.
- Title : `import export produits frais | Groupe Guanter Rodriguez`.
- H1 : spécialiste import export produits frais.
- Forces : spécialisation agroalimentaire/frais.
- Faiblesses : pas centré sur Garden Expresse : freight forwarding Maroc, affrètement, commissionnaire, corridors Maroc-Europe/Afrique.

### lkw-walter.com

- Statut : 200.
- Title : `The European Transport Organisation - LKW WALTER (UK)`.
- H1 : `The European Transport Organisation`.
- Forces : autorité, transport européen, FTL.
- Faiblesses : requêtes locales FR/MA moins ciblées. Garden Expresse doit éviter de les affronter sur termes génériques mondiaux, viser corridor+service+local.

## Opportunités Garden Expresse

1. Battre les EMD faibles par une architecture plus structurée.
2. Créer des pages `commissionnaire`, `transitaire`, `freight forwarder` non encore couvertes.
3. Renforcer Casablanca/Maroc dans schema + contenu + Google Business Profile.
4. Créer des pages de conversion `trouver camion`, `devis transport`, `affrètement camion`.
5. Produire des pages sectorielles fruits/légumes/agrumes avec preuves réelles.
6. Ne pas copier les concurrents : se différencier par clarté, preuve, rapidité de contact et contenu expert.
