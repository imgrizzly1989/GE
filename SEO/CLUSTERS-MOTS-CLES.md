# CLUSTERS MOTS-CLÉS — Garden Expresse

## Principe d'architecture

Garden Expresse doit viser une architecture **hub → clusters → pages transactionnelles** :

1. Hubs business : transport international, commissionnaire, freight forwarding, affrètement.
2. Hubs services : frigorifique, bâché, FTL, LTL, groupage, palettes.
3. Hubs corridors : Maroc-Europe, Europe-Maroc, Maroc-Afrique.
4. Pages corridor P0 : France, Espagne, Europe.
5. Pages locales : Casablanca + villes de départ/arrivée.
6. Pages conversion : devis, trouver camion, trouver transporteur.

## Règles de contenu

- Une page = une intention principale.
- Pas de duplication ville/corridor avec simple remplacement de noms.
- Ajouter données utiles : types de marchandises, ports/axes, délais indicatifs si vérifiables, documents, contraintes, FAQ.
- Toujours inclure contact, devis, téléphone, WhatsApp.
- Toujours éviter les promesses de position Google.
