## 2. Données macro fiables

### Commerce Maroc–UE

D’après la Commission européenne :

- L’UE est le **premier partenaire commercial du Maroc**.
- En 2025, l’UE représente **33,7 %** du commerce de biens du Maroc.
- **33,2 %** des exportations marocaines vont vers l’UE.
- **34 %** des importations marocaines viennent de l’UE.
- Commerce total de biens UE–Maroc 2025 : **62,2 Md€**.
- Importations UE depuis le Maroc : **25,5 Md€**.
- Exportations UE vers le Maroc : **36,7 Md€**.

Catégories principales côté UE :

| Sens du flux | Catégories dominantes selon Commission UE |
|---|---|
| UE importe depuis Maroc | machines & matériel de transport, produits agricoles, textiles |
| UE exporte vers Maroc | machines & matériel de transport, produits minéraux, food/raw materials |

### Commerce global Maroc

D’après World Bank API :

| Indicateur | 2024 |
|---|---:|
| Exports marchandises Maroc | ~45,226 Md USD |
| Imports marchandises Maroc | ~75,699 Md USD |
| Exports goods & services | ~67,454 Md USD |
| Imports goods & services | ~80,609 Md USD |
| Agriculture/forestry/fishing value added | ~10,57 % du PIB |

Implication : le Maroc a un fort volume import/export, mais importe plus de marchandises qu’il n’en exporte. Le SEO doit donc cibler **deux parcours différents** :

1. Exportateurs marocains qui envoient vers Europe/Afrique.
2. Importateurs marocains qui achètent depuis Europe.

## 3. Positionnement fruits & légumes marocains

### Produits export pertinents pour Garden Expresse

Les tendances sectorielles montrent une forte visibilité du Maroc sur :

- tomates fraîches ;
- tomates sous serre ;
- agrumes : mandarines, oranges, citrons, pamplemousses ;
- fruits rouges : myrtilles/blueberries, framboises, fraises ;
- légumes frais : haricots verts, poivrons, concombres, maïs doux ;
- avocats et autres produits frais selon saisons ;
- produits agroalimentaires conditionnés ;
- produits maritimes, à traiter séparément si Garden Expresse veut ce marché.

### Contraintes logistiques communes

Pour le transport fruits/légumes, les pages SEO doivent traiter :

- température dirigée ;
- frigorifique ;
- délai de transit ;
- rupture de froid ;
- disponibilité camion en saison ;
- groupage compatible uniquement si produits/températures compatibles ;
- palettes, poids, volume, emballage ;
- départs Agadir, Souss-Massa, Casablanca, Marrakech, Tanger, Larache/Kénitra selon produit ;
- passages Tanger Med / Espagne ;
- documents export ;
- coordination Morocco Foodex / contrôles / documents phytosanitaires quand applicable, sans prétendre être l’autorité ou déclarant.

## 4. Import Maroc depuis Europe

### Pourquoi c’est stratégique

Le Maroc importe massivement depuis l’UE : machines, matériel de transport, produits minéraux, matières premières, denrées/produits alimentaires et biens industriels.

Pour Garden Expresse, cela signifie que le SEO ne doit pas parler seulement d’export agricole. Il faut aussi capter :

- importateurs marocains ;
- PME marocaines qui achètent en France, Espagne, Italie, Allemagne, Belgique, Pays-Bas ;
- transport Europe → Maroc ;
- groupage Europe Maroc ;
- camion complet Europe Maroc ;
- transport palettes Europe Maroc ;
- transport machines Europe Maroc ;
- transport pièces industrielles Europe Maroc ;
- transport alimentaire Europe Maroc ;
- dédouanement / documents, avec prudence juridique.

## 7. Risques éditoriaux à éviter

Ne pas écrire :

- “leader / numéro 1 export Maroc” sans preuve ;
- “transport garanti en X jours” sans validation opérationnelle ;
- “certifié Foodex” si Garden Expresse n’a pas cette certification ;
- “dédouanement complet” si l’entreprise n’a pas l’agrément ou le partenaire explicite ;
- “tous produits frais compatibles groupage” — risque chaîne du froid et contamination ;
- chiffres produits exacts sans source officielle ou citation claire.

Écrire plutôt :

- “organisation de transport” ;
- “coordination avec les intervenants du dossier” ;
- “selon disponibilité camion et documents” ;
- “température dirigée selon cahier des charges” ;
- “cotation après validation trajet, marchandise, poids, volume, date et contraintes”.

## 10. Conclusion

Le potentiel SEO import/export est réel et plus défendable que des pages génériques. Le Maroc exporte fortement vers l’Europe, notamment via produits agricoles/agroalimentaires, tandis que le Maroc importe massivement depuis l’UE des biens industriels, machines, équipements et matières/produits alimentaires.

Pour Garden Expresse, le cluster gagnant doit combiner :

- export fruits/légumes Maroc Europe ;
- import Europe Maroc ;
- transport frigorifique ;
- documents import/export ;
- devis transport international ;
- corridors concrets France/Espagne/Italie/Allemagne/Benelux.

Prochaine étape recommandée : créer les 7 pages Phase A, puis contrôler indexation/canonicals/sitemap avant d’aller plus loin.
