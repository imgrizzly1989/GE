Date : 2026-06-22  
Projet : Garden Expresse  
Axe SEO : Europe → Maroc, c’est-à-dire entreprises européennes qui veulent expédier au Maroc.

## Sources fiables utilisées

1. Commission européenne — EU trade relations with Morocco  
   https://policy.trade.ec.europa.eu/eu-trade-relationships-country-and-region/countries-and-regions/morocco_en

2. World Bank API — Morocco merchandise imports/exports  
   - `TM.VAL.MRCH.CD.WT` : imports marchandises Maroc
   - `TX.VAL.MRCH.CD.WT` : exports marchandises Maroc

3. Rapport Garden Expresse déjà créé : `SEO/RECHERCHE-IMPORT-EXPORT-MAROC.md`

## Base macro

Selon la Commission européenne :

- L’UE est le premier partenaire commercial du Maroc.
- En 2025, 34 % des importations marocaines viennent de l’UE.
- Les exportations de l’UE vers le Maroc atteignent 36,7 Md€.
- Les catégories dominantes côté UE → Maroc sont :
  - machines et matériel de transport : 14,9 Md€ / 40,5 % ;
  - produits minéraux : 4,8 Md€ / 13 % ;
  - food and raw materials : 4,3 Md€ / 13 %.

Selon World Bank API :

- Imports marchandises Maroc 2024 : environ 75,699 Md USD.
- Exports marchandises Maroc 2024 : environ 45,226 Md USD.

Conclusion : le Maroc est un marché importateur important, et l’axe Europe → Maroc mérite un cluster SEO complet, pas seulement une page générique.

## Angle de contenu à utiliser

Chaque page doit :

- parler à l’exportateur européen ET à l’importateur marocain ;
- expliquer les informations nécessaires pour devis : départ, arrivée, incoterm, poids, volume, palettes, marchandise, date ;
- proposer FTL, LTL, groupage selon volume ;
- mentionner France/Espagne/Italie/Allemagne/Benelux quand pertinent ;
- rester prudent sur douane : “coordination des informations/documents”, pas “dédouanement garanti” ;
- CTA WhatsApp/devis immédiat.

## Garde-fous

Ne pas écrire :

- “leader Europe Maroc” ;
- “numéro 1” ;
- “dédouanement inclus” sans preuve ;
- “transport ADR/pharma/température garanti” sans capacité documentée ;
- chiffres par industrie sans source.

À écrire :

- “organisation de transport Europe → Maroc” ;
- “selon disponibilité camion et documents” ;
- “cotation après validation du trajet, marchandise, poids, volume et date” ;
- “groupage ou camion complet selon volume”.
